## 專案結構 File Structure

```
├─ public
└─ src
    │  App.js
    ├─ api           // 管理axios與api接口
    ├─ components    // 所有頁面與元件
    │  ├─ Navbar
    │  └─ ScenicSpot
    ├─ helpers       // 通用函式
    └─ store         // redux相關
        ├─ actions
        └─ reducers
```

## 專案啟動 How to run

```
> $ npm ci
```

```
> $ npm run start
```
